# Music Bot for discord

## About
Music Discord bot with Youtube integration

## Requeriments
- Python3 
- pip
- Docker

## Tools

### [yt-dlp](https://github.com/yt-dlp/yt-dlp)
Fork of [youtube-dl](https://github.com/ytdl-org/youtube-dl). Python Lib utilized for audiostream, video/audio download and catch youtube webpage infos/metadata.

### [discord-api](https://discord.com/developers/docs/quick-start/overview-of-apps)
Utilizing Discord API to interact with servers and user's messages.
